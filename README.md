# Secret Munchies



## About

You are a *spy* working in a fast food chain called **McMunchies** inside the *Pentagon*. 
Prepare food orders while listening to *secrets* that you may overhear. 
Send these secrets back to HQ using *morse code*. 
You can fail 3 times morse code/preparing orders before your contract gets **terminated**.

To make it easy for you to get started with GitLab, here's a list of recommended next steps.

## Controls

- WASD - Change camera
- A / D - Dot / Dash
- ESC - Main Menu / Skip Tutorial
- LMB - Interaction
- SPACE - Skip dialogue
